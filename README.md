# 📖 GT-QURANREADER - عارض القرآن الكريم

تطبيق متكامل ومجاني لعرض القرآن الكريم مع واجهة ويب حديثة وميزات متقدمة. يدعم التشغيل المباشر والتثبيت بعدة طرق.

[![GT-QURANREADER](gt-quranreader.png)
](https://github.com/SalehGNUTUX/GT-QURANREADER/blob/main/gt-quranreader.png?raw=true)
## 🌟 الميزات الرئيسية

- ✅ **عرض كامل للمصحف** - 604 صفحة مصورة عالية الجودة
- ✅ **بحث متقدم** - في السور والآيات بأسماء متعددة
- ✅ **تلاوة صوتية** - استماع للتلاوة مع تفعيل زر التشغيل
- ✅ **وضع ليلي/نهاري** - لحماية العين في الإضاءة المختلفة
- ✅ **تنقل سهل** - بين السور والأجزاء والصفحات
- ✅ **واجهة متجاوبة** - تعمل على جميع الأجهزة والشاشات
- ✅ **دعم كامل للغة العربية** - واجهة وتصميم عربي

## 🚀 طرق التثبيت والاستخدام

### 1. 📦 الإصدار المحمول (AppImage) - **مستحسن**

**مميزات AppImage:**
- ⚡ لا يحتاج تثبيت
- 🔒 لا يحتاج صلاحيات مدير
- 📁 تطبيق مستقل واحد
- 🖥️ يعمل على معظم توزيعات لينكس

**التثبيت:**
```bash
# تحميل آخر إصدار من صفحة Releases
# جعله قابلاً للتنفيذ
chmod +x GT-QURANREADER-*.AppImage

# التشغيل
./GT-QURANREADER-*.AppImage
```

**أو من الطرفية:**
```bash
wget https://github.com/SalehGNUTUX/GT-QURANREADER/releases/latest/download/GT-QURANREADER-x86_64.AppImage
chmod +x GT-QURANREADER-x86_64.AppImage
./GT-QURANREADER-x86_64.AppImage
```

### 2. 🛠️ التشغيل المباشر من المصدر

**للمستخدمين المتقدمين أو المساهمين:**

```bash
# استنسخ المستودع
git clone https://github.com/SalehGNUTUX/GT-QURANREADER.git
cd GT-QURANREADER

# التأكد من المتطلبات
python3 --version  # يجب أن يكون 3.6+

# التشغيل
./gt-quranreader.sh
```

### 3. 📥 التثبيت عبر Flatpak

**مميزات Flatpak:**
- 🛡️ آمن ومعزول
- 🔄 تحديثات تلقائية
- 📊 إدارة مركزية للتطبيقات

**التثبيت:**
```bash
# إضافة المستودع (إذا لزم الأمر)
flatpak remote-add --if-not-exists flathub https://flathub.org/repo/flathub.flatpakrepo

# تثبيت التطبيق
flatpak install https://github.com/SalehGNUTUX/GT-QURANREADER/releases/latest/download/gt-quranreader.flatpak

# التشغيل
flatpak run com.gnutux.gt-quranreader
```

### 4. ⚙️ التثبيت التقليدي

**للتثبيت الدائم في النظام:**

```bash
# نسخ الملفات
sudo cp gt-quranreader.sh /usr/local/bin/gt-quranreader
sudo cp gt-quranreader.desktop /usr/share/applications/
sudo cp gt-quranreader.png /usr/share/pixmaps/

# جعله قابلاً للتنفيذ
sudo chmod +x /usr/local/bin/gt-quranreader

# التشغيل من أي مكان
gt-quranreader
```

## 🎯 دليل الاستخدام

### التشغيل الأولي
1. **شغل التطبيق** بأي طريقة من الطرق أعلاه
2. **سيفتح المتصفح تلقائياً** على `http://localhost:8787`
3. **إذا لم يفتح تلقائياً**: افتح متصفحك واذهب إلى `http://localhost:8787`

### التحكم في الواجهة
- **➡️ الصفحة التالية**: السهم الأيمن أو زر `D`
- **⬅️ الصفحة السابقة**: السهم الأيسر أو زر `A`
- **🔍 البحث**: انقر على زر البحث وأدخل النص
- **🎵 التلاوة**: انقر على زر السماعة لتشغيل التلاوة
- **🌙 الوضع المظلم**: انقر على زر القمر للتبديل
- **📖 قائمة السور**: انقر على اسم السورة للانتقال المباشر

### إيقاف التطبيق
- **في الطرفية**: اضغط `Ctrl + C`
- **في واجهة التطبيق**: أغلق نافذة المتصفح
- **في AppImage**: أغلق نافذة المتصفح أو النافذة الطرفية

## 🐛 استكشاف الأخطاء وإصلاحها

### المشكلة: المتصفح لا يفتح تلقائياً
**الحل:**
```bash
# افتح المتصفح يدوياً على العنوان:
http://localhost:8787

# أو جرب منفذ مختلف:
http://localhost:8788
```

### المشكلة: المنفذ مشغول
**الحل:** التطبيق سيجد منفذاً تلقائياً بديلاً

### المشكلة:缺少 Python
**الحل:**
```bash
# أوبونتو/ديبيان
sudo apt update && sudo apt install python3

# فيدورا/سنتوس
sudo dnf install python3

# أرش لينكس
sudo pacman -S python
```

### المشكلة: أخطاء في الصور أو البيانات
**الحل:**
```bash
# حذف البيانات القديمة وإعادة التحميل
rm -rf ~/.local/share/gt-quranreader
./gt-quranreader.sh
```
